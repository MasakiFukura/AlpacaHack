# Writeup for hit-and-miss

## 概要

サーバーに対して文字列を送ると、正誤判定が返ってくる。
正規表現のマッチ結果をトライ＆エラーで、正解の文字列を1文字ずつ導き出す。
このように、判定結果などのヒントを利用して秘密情報を特定する手法を、オラクル攻撃という。

## 解析

フラグに対して正規表現をかけて、一致すれば`Hit!`と出力される。
`re.match`は先頭からの一致判定であるため、部分的に条件を当てていくことができる。

```python
    if re.match(pattern, FLAG):
        print("Hit!")
```

さまざまなパターンで正規表現を送信し、答えにたどり着く必要がある。

フラグの内容は、`re.fullmatch(r"Alpaca\{\w+\}", FLAG)`と書かれている。
`\w`は数字・小文字・大文字・アンダーバーの4種類である。

## Exploit

全パターンの文字を検索するのは膨大な時間がかかるため、二分探索で検索する。

今回、二分探索の方法は、以下のように行った。

1. 数字・小文字・大文字・アンダーバーの4種類を分類
2. 正規表現で範囲（例:`[A-M]`）を指定
3. 結果を見ながら範囲をだんだん絞っていき、文字を特定

この処理を1文字ずつ順番に行い、特定した。

## Flag

`Alpaca{Reg3x_Crossw0rd}`
