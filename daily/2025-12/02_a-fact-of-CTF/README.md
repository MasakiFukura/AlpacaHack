# Writeup for a fact of CTF

## 問題

>AlpacaHack で一番最初に完成したものの難易度調整により出題されなかった幻の問題 （運営コメント）

暗号化するためのPythonスクリプトと、暗号化後の値が与えられる。
元のフラグ文字列を復元する問題。

## 解法

各文字を`ord()`でユニコードの数値に変換し、対応する素数をその回数だけ掛け合わせている。

例：
"A" → 65 → 2^65
"B" → 66 → 3^66

すべて掛け合わせた値が暗号化後の文字列。

素数を利用して暗号化しているというところから、素因数分解を利用して解くと発想する必要がある。
各素数で割れる回数（指数）を求め、それを文字列に変換することで復元できる。

1. `ct` を先頭の素数（2）で割れるだけ割る → 回数が1文字目の `ord`
2. 次に3, 5, 7... と順に同様に処理
3. 得られた値を `chr()` に変換して文字列を復元

以下のコードで解ける。

```python
flag = ""

for p in primes:
    cnt = 0
    while crypt % p == 0:
        crypt //= p
        cnt += 1

    if cnt == 0:
        break

    flag += chr(cnt)

print(flag)
```
